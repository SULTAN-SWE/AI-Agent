# Enterprise AI Operations Platform

This folder contains a runnable local platform build that follows the PDF specification as a practical prototype.

## What is included

- `enterprise_platform_server.py` - local platform server and API
- `enterprise_platform.db` - persisted local data
- `enterprise-ai-operations-platform-suite.html` - front-end suite
- `Dockerfile` - container build
- `docker-compose.yml` - local container run

## Platform surfaces

- AI workspace and request routing
- Approvals and audit logging
- Notifications, memory, workflow runs, and insights
- System, roadmap, and agent metadata

## Run locally

```bash
python enterprise_platform_server.py
```

Then open:

```text
http://127.0.0.1:8010
```

## Run with Docker

```bash
docker compose up --build
```

Then open:

```text
http://127.0.0.1:8010
```

## Publish later

1. Copy this folder to a Linux server or deployment platform.
2. Expose port `8010` or put a reverse proxy in front of it.
3. Add HTTPS with a domain name.
4. Move the SQLite database to a managed production database when you are ready.
5. Replace the in-memory session store with a persistent session store for production.
